# 📊 Расчёт сметной стоимости ИГИ по НЗ

**Приказ Минстроя РФ №281/пр от 12.05.2025**

Приложение для расчёта сметной стоимости инженерно-геологических изысканий по нормативным затратам (НЗ).

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd "C:\Users\User\Documents\Antigravity work\Agents\smeta"
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
streamlit run app.py
```

Приложение откроется в браузере по адресу: **http://localhost:8501**

## 📁 Структура проекта

```
smeta/
```

## ⚙️ Конфигурация

### Обновление индекса цен

Индекс пересчёта к текущим ценам обновляется ежеквартально по данным Минстроя. Пока используется базовый уровень цен 01.01.2024 (индекс = 1.0).

## 🔧 Разработка

### Тестирование калькулятора

```bash
python modules/calculator.py
```

### Добавление новых видов работ

1. Добавьте расценку в `data/normative_costs.json`
2. Добавьте описание работы в `data/work_types.json`
3. Укажите связь через `base_cost` или `cost_key`

## 📝 Примечания

- Базовый уровень цен: **01.01.2024**
- НДС **не включён** в расценки
- Расчёт соответствует методике Приказа №1/пр от 09.01.2024
